# CleanLoo – NYC Public Restroom Finder

CleanLoo is a tool designed to help New York City residents and visitors locate clean, accessible public restrooms. It uses the NYC Open Data **Public Restrooms Dataset** to provide accurate information about restroom locations, accessibility, and year-round availability.

## Features
- Search for public restrooms by borough  
- Filter by ADA accessibility  
- Identify year-round vs. seasonal restrooms  
- View restroom details (location, type, accessibility)  
- Designed using structured data and algorithmic principles  

## Data Source
**NYC Open Data – Public Restrooms Dataset**  
https://data.cityofnewyork.us/City-Government/Public-Restrooms/i7jb-7jku

## How It Works
1. Load dataset into a `restroomList` structure  
2. Apply filters (borough, accessibility, open year-round)  
3. Return matching restrooms  
4. Display results through a prototype UI
